# 📦 StockMaster - Gestión de Inventario

![Android](https://img.shields.io/badge/Platform-Android-3DDC84?style=flat-square&logo=android)
![Firebase](https://img.shields.io/badge/Backend-Firebase-FFCA28?style=flat-square&logo=firebase)
![Kotlin](https://img.shields.io/badge/Language-Kotlin-7F52FF?style=flat-square&logo=kotlin)

**StockMaster** es una aplicación móvil robusta diseñada para el control de inventarios en tiempo real. Permite a los usuarios gestionar productos, stock y categorías de forma eficiente integrando la potencia de Firebase.

## ✨ Características

* **Autenticación en la Nube:** Registro e inicio de sesión seguro con Firebase Authentication.
* **Gestión en Tiempo Real:** CRUD (Crear, Leer, Actualizar, Eliminar) de productos conectado a Firestore.
* **Menú Lateral (Drawer):** Navegación fluida con acceso a perfil y secciones.
* **Diseño Moderno:** Interfaz basada en Material Design con iconos adaptativos.
* **Icono Personalizado:** Fácil identificación visual con el logo de StockMaster.

## 🛠️ Tecnologías utilizadas

* **Lenguaje:** [Kotlin](https://kotlinlang.org/)
* **Base de Datos:** [Cloud Firestore](https://firebase.google.com/docs/firestore)
* **Autenticación:** [Firebase Auth](https://firebase.google.com/docs/auth)
* **UI Components:** Material Design Components, Navigation Drawer, RecyclerView.
